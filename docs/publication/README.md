# Publication Shelf

This shelf contains all publication planning, release preparation, and export materials for ADOP.
Files here are internal planning docs — they are not themselves proof that ADOP is already public.

## Reading Order

Start from the top if unfamiliar. Jump to the relevant section if returning to a specific task.

### 1. Context

- `STANDALONE_CANDIDATE_NOTE.md` — confirms this directory is the standalone ADOP project root

### 2. Open Decisions (resolve before any release action)

- `PRE_PUBLICATION_DECISIONS.md` — outstanding decisions that block publication
- `OWNER_DECISION_PACKET.md` — full owner decision checklist: license, host, contribution policy, security intake

### 3. Readiness Verification

- `PUBLISHABLE_HOLD_CHECKLIST.md` — verify the repository is in publishable hold state
- `RELEASE_BOOTSTRAP_NOTES.md` — notes to review just before standalone repository creation

### 4. Export

- `REPO_EXPORT_GUIDE.md` — defines standalone repository shape and carry/no-carry file mapping
- `REPO_EXPORT_CHECKLIST.md` — step-by-step pre-extraction checklist

### 5. Promotion

- `PUBLIC_POLICY_PROMOTION_MATRIX.md` — how to promote draft documents to the public surface

### 6. Public Release Track

- `PUBLIC_VERIFICATION_CONTRACT.md` — required verification before publication
- `PUBLIC_GO_LIVE_SEQUENCE_DRAFT.md` — ordered sequence for actual public release
- `PUBLICATION_RUNBOOK.md` — executable runbook from publishable hold to public release

### 7. Private Bootstrap Track (alternative: prepare private repo first)

- `PRIVATE_HOST_BOOTSTRAP_RUNBOOK.md` — execution order for private repository setup
- `PRIVATE_RELEASE_HOLD_LINE.md` — explicit stop point after private preparation

## Stop Rule

Do not treat any file in this shelf as evidence that ADOP is already public.
All publication actions require the owner to explicitly complete `OWNER_DECISION_PACKET.md` first.
